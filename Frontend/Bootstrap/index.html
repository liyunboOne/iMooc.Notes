'form-group': 代表是一个组件

  1. Bootstrap中一些怪异的属性
  2. role: 出自html5，是给特定浏览器工具使用，如盲文
  3. aria-label: 配合输入框使用，当焦点落在输入框，读屏软件会读出输入框的内容
  4. tabIndex: 控制焦点在输入框中移动的顺序
  5. data-???: html5中新增属性，太熟悉了
  1. 组件相关
  2. data-toggle: 绑定相关事件
  3. .input-group: 控件组，可以在控件组中添加图标
  4. .input-group-addon: 可放置额外内容及图标
